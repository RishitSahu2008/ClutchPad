## What's clutchpad??
Clutchpad is a macropad that is designed to increase your efficiency in tasks and offer a clean, cool look to your setup. It feateures - 
- 9 switches arranged in 3*3 matrix layout
- 6 LED's so you never miss the glow
- an oled display (for funk)
- a rotary encoder for controlling volume
- seed studio's xiao rp2040 microcontroller

And the one major thing it features is customisability. You can change what each key does, how the LED's light up, what to display on the oled screen and even the rotary encoder. 

I made this one for gaming so the keys are assigned accordingly.

## Firmware 
This project uses KMK but i also plan to add QMK later in the future. 
Also curently only one layer is there but i will add further more layers so i can use this for multiple tasks. 

## Challenges faced
I had been out of touch with pcb designing for a long time so it was quite timetaking to realign myself. Also i was new to 3d modelling but it was not too tough to understand. The main thing was the firmware . It took a lot of time as the documentation was not quite undertandable and also there was little content on outube. I tried switching to QMK but found it problematic with the rotary encoder and the display, which the KMK firmware has libraries of.
